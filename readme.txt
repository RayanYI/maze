Ce fichier a pour objectif de vous aider a utiliser correctement et a bien comprendre les différents modules de notre projet.
Le module graphe contient la classe Graphe qui modélise des graphes à partir de dictionnaires d'adjacence et de poids, ainsi que de méthodes essentielles liées au graphes.
Le module render contient la classe Render qui utilise matplotlib afin d'afficher les différents objets utilisés dans le projet.
Le module generation_grille contient la classe Generation qui permet de créer des graphes correspondant à des grilles qui peuvent être pondérées ou biaisées.
Le module algo_Prim.py contient la classe Algo qui permet de générer un arbre couvrant du graphe d'une grille pondérée à partir de l'algorithme de Prim.
Le module resolution contient la classe Resolution qui permet de générer un chemin affichable résolvant un labyrinthe donné.
Le module test contient quelques exemples de tests que nous avons opéré afin de vérifier que les codes que nous avons créés marchaient.
Le module main sert à générer un labyrinthe et c'est le module à utiliser dans le cadre d'une utilisation dite classique, c'est-à-dire pour laquelle nous avons pensé notre projet
